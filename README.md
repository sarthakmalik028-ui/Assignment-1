# Assignment-1

Task 1:
This program is a basic calculator that performs four arithmetic operations on two user-entered numbers.
How It Works:

The program prompts the user to enter two numbers.

It stores the numbers in variables:

a → first number

b → second number

It performs the following operations:

Addition → a + b

Subtraction → a - b

Multiplication → a * b

Division → a / b

It prints the results of all four operations.



Task 2:
This program is a simple greeting program that takes the user's name as input and displays a personalized welcome message.
How It Works

The program asks the user to enter:

Their first name

Their last name

It stores the inputs in variables:

First_Name

Last_name

It combines (concatenates) the first and last name into a greeting message.

It prints a personalized welcome message.
